# دليل تدفقات النظام (System Flows Guide)

هذا المجلد يحتوي على جميع مخططات التدفق (Flowcharts) والشرح التفصيلي لعمليات النظام الرئيسية في منصة TagDoD للتجارة الإلكترونية.

## 📋 فهرس التدفقات

### 🔐 المصادقة وإدارة المستخدمين
- [**المصادقة**](./auth/) - تسجيل المستخدمين وتسجيل الدخول
  - `signup-login.mmd` - مخطط تسجيل/دخول المستخدمين
- [**الأدوار والصلاحيات**](./rbac/) - إدارة الأدوار والتحكم في الوصول
  - `role-permissions.mmd` - مخطط RBAC والتحقق من الصلاحيات

### 🛍️ الكتالوج والبحث
- [**الكتالوج**](./catalog/) - تصفح المنتجات والبحث
  - `search-browse.mmd` - مخطط تصفح الكتالوج والبحث
- [**إدارة المنتجات**](./products/) - إدارة المنتجات والتنويعات
  - `product-management.mmd` - مخطط إدارة المنتجات

### 🏠 العناوين والمفضلات
- [**إدارة العناوين**](./addresses/) - إدارة عناوين العملاء
  - `address-management.mmd` - مخطط إدارة العناوين
- [**المفضلات**](./favorites/) - إدارة المفضلات ومزامنتها
  - `favorites-sync.mmd` - مخطط المفضلات والمزامنة

### 📁 الوسائط والملفات
- [**الوسائط**](./media/) - إدارة الملفات والصور
  - `media-upload.mmd` - مخطط رفع وإدارة الوسائط

### 🛒 السلة والطلبات
- [**السلة**](./cart/) - إدارة سلة التسوق ومزامنة الزوار
  - `add-sync.mmd` - مخطط إضافة العناصر ومزامنة السلة
  - `abandoned-cart.mmd` - مخطط استرداد السلات المهجورة

- [**الشيك آوت**](./checkout/) - عملية الشراء الكاملة
  - `checkout-flow.mmd` - مخطط خطوات الشيك آوت

- [**الطلبات**](./order/) - إدارة حالات الطلبات
  - `status-fsm.mmd` - آلة حالات الطلبات

### 💳 الدفع والمخزون
- [**الدفع**](./payment/) - معالجة المدفوعات
  - `webhook.mmd` - مخطط معالجة Webhook الدفع

- [**المخزون**](./inventory/) - إدارة المخزون والحجوزات
  - `adjustments.mmd` - مخطط تعديلات المخزون

### 🎫 الكوبونات والعروض
- [**الكوبونات**](./coupons/) - تطبيق وإدارة الكوبونات
  - `coupon-application.mmd` - مخطط تطبيق الكوبونات

### 📞 الدعم والإشعارات
- [**الدعم الفني**](./support/) - نظام تذاكر الدعم
  - `ticket-flow.mmd` - مخطط تدفق تذاكر الدعم

- [**الإشعارات**](./notifications/) - نظام الإشعارات متعدد القنوات
  - `notification-system.mmd` - مخطط نظام الإشعارات

### 📊 التقارير والتحليلات
- [**التقارير**](./reports/) - توليد التقارير والإحصائيات
  - `report-generation.mmd` - مخطط توليد التقارير

## 🎯 كيفية الاستخدام

### عرض المخططات
1. افتح أي ملف `.mmd` في محرر يدعم Mermaid
2. أو استخدم [Mermaid Live Editor](https://mermaid.live)
3. أو استخدم أي عارض Mermaid في IDE

### فهم التدفقات
1. ابدأ بقراءة ملف `README.md` في كل مجلد
2. راجع المخطط `.mmd` المقابل
3. ارجع للوثائق التقنية في `docs/database/` و `docs/requirements/`

## 📝 ملاحظات مهمة

- جميع المخططات مبنية على الكود المصدري الفعلي للمشروع
- التدفقات تعكس العمليات الحقيقية في النظام
- تم تحديث المخططات لتتماشى مع أحدث إصدار من النظام

## 🔄 التحديثات

عند تعديل أي تدفق في الكود:
1. حدث المخطط المقابل في `.mmd`
2. حدث الشرح في `README.md`
3. حدث قائمة التدفقات في `flows-checklist.md`
